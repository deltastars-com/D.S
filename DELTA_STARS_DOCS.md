# دليل تشغيل ونظام متجر نجوم دلتا للتجارة (النظام السيادي الموحد) 🌟

هذا المستند يمثل الدليل التقني والتشغيلي الشامل لمتجر "نجوم دلتا" بعد إتمام كافة التحسينات التقنية والأمنية لعام 2026.

## 1. آلية عمل المتجر (Store Mechanism & Workflow)
يعمل المتجر بنظام هجين يدمج بين التجارة الإلكترونية التقليدية وإمدادات المؤسسات (B2B/B2C):
1. **تجربة المستخدم**: واجهة ذكية متوافقة مع كافة الشاشات والمتصفحات، تدعم اللغتين العربية والإنجليزية.
2. **محرك البيانات السيادي**: نظام معالجة لـ **240 صنفاً** حقيقياً مرتبطاً بأيقونات الأقسام بديناميكية عالية.
3. **ذكاء الأوزان**: نظام ذكي للتحويل بين الكيلو جرام (بدءاً من 500 جرام لضمان المرونة) والعبوات (باكت) والحزم (Hizmah) لضمان دقة الطلب.
4. **تدفق الطلبات**: يمر الطلب عبر "محلل سيادي" يوجهه للمستودع أو المندوب الأنسب جغرافياً.

## 2. التقنيات والبروتوكولات (Tech Stack & Protocols)
*   **اللغة**: TypeScript (لضمان أمان الأكواد ومنع الأخطاء البرمجية).
*   **الواجهة**: React 18+ مع Tailwind CSS (التوافقية العالية والسرعة الفائقة).
*   **قاعدة البيانات**: Firebase Firestore (تحديثات لحظية/Real-time).
*   **الأمان**: 
    *   بوابة الدخول السيادية (Sovereign Login).
    *   نظام التحقق الحيوي (Biometric WebAuthn) لقسم المطور.
    *   رمز الوصول الجذر (Root Access Code: 321666).
*   **PWA**: تطبيق ويب تقدمي قابل للتثبيت كـ App ثابت بفاعلية Android/iOS.
*   **SEO**: تحسين كامل لمحركات البحث ببيانات تعريفية (Meta Description) دقيقة.

## 3. المميزات الحصرية (Key Features)
*   **المساعد الذكي (عدي AI)**: مساعد مدمج لتحليل رغبات العميل والمساعدة في اختيار أفضل المنتجات.
*   **صالة العرض (Showroom)**: تجربة تسوق غامرة مع فلاتر ذكية لكل فئة (خضروات، فواكه، تمور، مستورد، موسمية).
*   **تتبع الأسطول المبرد (FleetRadar)**: نظام تتبع حي للمناديب لضمان وصول المنتجات طازجة.
*   **بوابة VIP**: مخصصة للعملاء الدائمين والمؤسسات بخصومات مبرمجة.

## 4. إدارة المطور (Developer Sovereign Console)
يعد قسم المطور مركز التحكم الرئيسي، ويشمل:
*   **إدارة الأكواد**: تحديث مباشر لمنطق التسعير والوزن.
*   **إدارة المخزون**: تحديثات لحظية لـ 235+ منتج.
*   **تحليلات سيادية**: تقارير مالية وتشغيلية متقدمة.

## 5. الأوزان والوحدات الملكية (Weight & Unit Logic)
تم ضبط النظام ليلائم السوق السعودي بدقة:
*   **الوزن الحر**: تبدأ منتجات الوزن من 500 جرام (نصف كيلو) آلياً عند اختيار الكيلو.
*   **الحزم والباكت**: يتم التعامل مع الحزمة كأصغر وحدة لمنتجات الورقيات والعبوات المغلفة لضمان جودة التغليف.

## 6. الجاهزية والنشر (Deployment Readiness)
*   المتجر خالي من أخطاء "Contrast" و"Accessibility".
*   متوافق مع سياسات Google Play و Apple App Store.
*   جاهز للتحويل إلى ملف APK/IPA عبر Capacitor/Cordova.
